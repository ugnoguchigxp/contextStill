# Support

## Questions and Usage Help

Use GitHub Issues for:

- Setup problems.
- Documentation gaps.
- Reproducible bugs.
- Feature requests with concrete use cases.

Please include:

- OS and Bun version.
- Commit SHA or release version.
- Relevant command output.
- Whether Docker, PostgreSQL, embedding, LLM, and search providers are configured.
- `bun run doctor` output with secrets removed.

## Security Reports

Do not file public issues for vulnerabilities. See [SECURITY.md](SECURITY.md).

## Scope

context-still is local-first infrastructure. Maintainers may decline support for custom hosting, exposed public deployments, private provider configuration, or local data recovery that cannot be reproduced.
